# create-react-app

使用 create-react-app 创建一个 React 项目

```sh
# 快速创建项目
$ npm i -g create-react-app
$ create-react-app my-app
$ cd my-app/
$ npm start 启动本地服务器
$ npm run build 压缩代码
```
## 自定义配置文件

覆盖 create-react-app webpack 配置 不需要 eject 使用 [`react-app-rewired`](https://github.com/timarney/react-app-rewired)
