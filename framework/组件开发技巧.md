## 弹窗

穿透问题：禁止下面层级页面滚动，弹出窗口时给 body 增加样式：

```js
.frozen {
    position: fixed;
    width: 100%;
}
```

阻止浏览器的滑动问题：

```js
document.addEventListener('touchmove', hendler, {passive : false});
```
