## 自定义 Taro 组件 与原生组件交互

Taro 组件自定义事件

```js

onChange = () => {
   this.$scope.triggerEvent('select', departmentInfo)
}


<View onClick={this.onChange}></View>
```

原生组件调用

```js
  onChange: function(event) {
    this.queryWayBillInfoByCondition(event.detail)
  },
  
  <custom-compoent bindselect="onChange"></custom-compoent>
```
