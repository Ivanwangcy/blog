## 安装 OSX 的React Native 环境
```sh
$ brew update && brew doctor
# 根据提示安装
$ xcode-select --install # xcode 命令行工具

$ brew install watchman
$ brew install flow

$ npm install react-native-cli -g

$ react-naive run-ios
```
