## 常用函数

### isEmpty


可以判断空对象，空数组

```js
import isEmpty from 'lodash/isEmpty'

 console.log('====================================');
 console.log('isEmpty({})',isEmpty({})); // true
 console.log('isEmpty([])',isEmpty([])); // true
 console.log('====================================');
```
