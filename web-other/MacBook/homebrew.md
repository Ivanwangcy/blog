# HomeBrew OS X 不可或缺的软件管理器
## 安装 Homebrew
```sh
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
## brew 常用命令
```sh
$ brew --version  # 查看版本
$ brew update # 更新 Homebrew 自身

```

## 使用 homebrew 安装 `node`
```sh
$ brew install node
```
## 查找可安装的软件
```sh
$ brew search --desc <keyword>
```

## 帮助文档
```sh
$ brew help
$ man brew 
$ brew doctor # 修复
```
