# Oh-My-zsh 命令行操作
# 配置 profile

## Oh-My-zsh git
```sh
#### 常用
$ gst # 等于 git status 查看状态
$ gss # 等于 git status -s 简化的 git 状态
$ gcmsg "message" # 等于 git commit -m "说明"
$ gaa # 等于 git add --all
$ gp # 等于 git push
$ gl # 等于 git pull
$ gco # 等于 git checkout
$ glg # 等于 git log --stat --max-count = 10
$ glog # 等于 git log --oneline --decorate --color --graph


```
