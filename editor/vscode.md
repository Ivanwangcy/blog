## vscode 使用
### 快捷键

### 项目管理
- `Ctrl + K Ctrl + O` 打开项目文件夹
- `Ctrl + K F` 关闭当前项目
- `Ctrl + Shift + P` 命令面板
### 视图操作
- `Ctrl + +/-` 放大或缩小视图
- `Ctrl + ↑/↓` 向上/下移动可视区域
- `Ctrl + c / Esc` 进入 Vim 模式
### 行操作
- `Alt + ↑/↓` 向上/下移动当前行
- `Shift + Alt + ↑/↓` 向上/下复制当前行
