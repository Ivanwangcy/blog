## vscode 使用

按住 Alt 键，点击变量可以跳转到声明的位置或引用的文件。与 Atom 的 hyperclick 一样。

### 快捷键

### 项目管理

- `Ctrl + K Ctrl + O` 打开项目文件夹
- `Ctrl + K F` 关闭当前项目
- `Ctrl + Shift + P` 命令面板

### 视图操作

- `Ctrl + +/-` 放大或缩小视图
- `Ctrl + ↑/↓` 向上/下移动可视区域
- `Ctrl + c / Esc` 进入 Vim 模式


### 行操作

- `Alt + ↑/↓` 向上/下移动当前行
- `Shift + Alt + ↑/↓` 向上/下复制当前行

## 常用插件 

* :sync 同步代码编辑器配置。

* :prettier
  - ctrl + alt + B 格式化当前文件;

* [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) 可以查看每一行的最新代码提交人
