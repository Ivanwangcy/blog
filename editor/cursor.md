## 快捷键

- cmd + p 查找/打开文件
- cmd + shift + p 打开命令面板
- cmd + shift + n 打开新窗口
- cmd + shift + w 关闭窗口
- cmd + shift + t 打开标签页
- cmd + shift + f 打开搜索

- shift + option + ⬆️ 向上复制行
- shift + option + ⬇️ 向下复制行
- option + ⬆️ 向上移动
- option + ⬇️ 向下移动
