# blog
  我的markdown博客.<br>
推荐使用 `Atom` 编辑器来写MD文件. 支持即时预览效果, 非常好用.<br>
## 首先, 打开要编辑的文件
---
按`Ctrl+Shift+M`, 打开MD文件预览界面. 如果无法呼出, 可以使用`Ctrl+Shilt+P`,搜索 markdown Preview 窗口. 写完别忘了,提交到GitHub上. :ok::ok_hand:!
