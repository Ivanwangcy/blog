## 首先, 打开要编辑的文件

---

按`Ctrl+Shift+M`, 打开 MD 文件预览界面. 如果无法呼出, 可以使用`Ctrl+Shilt+P`,搜索 markdown Preview 窗口. 写完别忘了,提交到 GitHub 上. :ok::ok_hand:!
